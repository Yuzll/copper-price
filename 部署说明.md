# 铜价自动更新系统 — 部署说明

## 这是什么？

一个基于 GitHub Actions 的**全自动铜价爬虫**。每天北京时间 08:00 自动从长江有色金属网抓取 1#电解铜报价，发布为 JSON 格式。

你的机电造价工具会自动拉取这个 JSON，**用户打开工具即可看到最新铜价**，无需手动输入。

---

## 🚀 部署步骤（5分钟）

### 第1步：注册 GitHub 账号

访问 https://github.com 注册（免费），记好你的**用户名**。

### 第2步：创建仓库

1. 登录后点击右上角 **+** → **New repository**
2. Repository name 填：`copper-price`（或你喜欢的名字）
3. 选择 **Public**（公开，免费）
4. **不要**勾选 "Add a README file"
5. 点击 **Create repository**

### 第3步：上传文件

你会看到 "Quick setup" 页面。**复制仓库地址**（形如 `https://github.com/你的用户名/copper-price.git`）。

然后在电脑上打开终端，进入 `github-copper-crawler` 目录，运行：

```bash
# 初始化 git
git init
git add .
git commit -m "初始提交：铜价自动爬虫"

# 关联你的仓库（替换为你的地址）
git remote add origin https://github.com/你的用户名/copper-price.git

# 推送
git branch -M main
git push -u origin main
```

> 💡 **如果不会用命令行**：在 GitHub 网页上，直接拖拽 `github-copper-crawler` 文件夹里的所有文件到仓库页面也可以。

### 第4步：启用 GitHub Pages（关键！）

1. 进入仓库 → **Settings** → 左侧点 **Pages**
2. "Source" 选 **Deploy from a branch**
3. "Branch" 选 **main**，文件夹选 **/ (root)**
4. 点击 **Save**
5. 等1-2分钟，页面顶部会出现：
   > ✅ Your site is live at `https://你的用户名.github.io/copper-price/`

### 第5步：测试

1. 进入仓库 → **Actions** 标签
2. 点击左侧 **每日更新铜价**
3. 点击 **Run workflow** → **Run workflow**（手动触发一次）
4. 等30秒，看到 ✅ 绿色勾就是成功了

### 第6步：获取你的 JSON 地址

你的铜价数据地址是：
```
https://你的用户名.github.io/copper-price/copper.json
```

例如用户名是 `zhangsan`：
```
https://zhangsan.github.io/copper-price/copper.json
```

**把这个地址记下来，下一步要填入工具。**

## 🔧 自定运行时间

如果需要改定时时间，编辑 `.github/workflows/update-copper.yml`：

```yaml
schedule:
  - cron: '0 0 * * *'   # UTC 时间，北京时间 = UTC+8
```

| cron 表达式 | 北京时间 |
|------------|---------|
| `0 0 * * *` | 每天 08:00 |
| `0 22 * * *` | 每天 06:00 |
| `30 1 * * *` | 每天 09:30 |

> ⚠️ GitHub Actions 免费额度：每月 2000 分钟，这个爬虫每月只用约5分钟，完全够用。

## ❓ 常见问题

**Q: 爬虫失败了怎么办？**
A: GitHub 会自动发邮件通知你。失败通常是因为网站改版，去 https://www.cjys.net/price 看看页面结构是否变了。

**Q: 铜价数据多久更新一次？**
A: 长江有色金属网交易日更新（周一至周五），周末和节假日价格不变。

**Q: 用户能看到我的 GitHub 仓库吗？**
A: 能，但无所谓——铜价本来就是公开数据，你的仓库只是一个搬运工。
